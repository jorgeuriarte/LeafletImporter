    🔑 How to create an App Password

    App Passwords are special keys that allow you to connect applications without using your main password.

    1. Go to bsky.app/settings/app-passwords
    2. Click on "Add App Password"
    3. Give it a name (e.g., "Leaflet Importer")
    4. Copy the generated key (format: xxxx-xxxx-xxxx-xxxx)
    5. Paste it in the field above

    Note: You can revoke this key at any time from the same settings page.
